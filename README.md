## OPPO Find X (PAFM00) TWRP Builder

## Support Device

1. OPPO Find X
2. devcode: PAFM00
3. TARGET: Recovery
4. MAIN REPO: TWRP-11
5. ANDROID 11

---

### [Download](https://github.com/qwe28256/Findx-11-TWRP-Builder/releases)

---

## Bugs

1. Touch Screen

## How to use

1. Unlock The Devies

```
fastboot flashing unlock
```
2. Flash Recovery

```
fastboot flash recovery <form-release>.img
```

3. Select To Recovery && Reboot
